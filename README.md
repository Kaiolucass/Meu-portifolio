## 🏗️ Estrutura do Projeto

O site foi construído focando em performance e interatividade:

- Bootstrap 5: Para um layout responsivo e moderno.

- AOS (Animate On Scroll): Para animações fluidas durante a navegação.

- tsParticles: Efeitos visuais de fundo para uma experiência imersiva.

- Vlibras: Acessibilidade garantida para todos os usuários.

- Modo Noturno Nativo: Implementado via Bootstrap data-bs-theme.
